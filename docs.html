Comment ça marche ?

Soliguide Tools est une boîte à outils interne pour croiser les données Soliguide avec des sources publiques (data.gouv.fr, FINESS, subventions…). L'objectif : identifier les structures absentes de Soliguide, mesurer la couverture par territoire, et enrichir la base.

1

Source publique

Les scripts TypeScript téléchargent les données depuis data.gouv.fr, des API publiques, ou des fichiers CSV.

2

Traitement & matching

Les scripts nettoient, transforment et croisent les données avec Soliguide. Le résultat est exporté en JSON.

3

Dashboard interactif

Le dashboard HTML/JS affiche les résultats : cartes, tableaux, statistiques. Pas de build, pas de framework.

🤖

Piloté par Claude Code

Tu n'as pas besoin de coder. Ouvre un terminal, tape claude, et décris ce que tu veux en français. L'assistant crée les scripts, le pipeline de données et le dashboard. Les skills (commandes slash) accélèrent le processus.
