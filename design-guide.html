Layout obligatoire

Header (navbar)

  • Fond blanc, shadow, border-b
  • Hamburger menu à gauche (lien accueil en premier, puis les autres projets)
  • Logo Soliguide centré en couleurs naturelles (pas de brightness-0 invert)
  • Nom de l'outil en text-primary
  • Filtres globaux à droite

Footer

  • Fond bg-primary texte blanc
  • Symbole Soliguide inversé (brightness-0 invert, opacity-50)
  • Crédits source de données
  • Lien "← Retour à l'accueil" aligné à droite

Body

  • class="min-h-screen bg-base-100 flex flex-col"
  • Main en flex-1 max-w-7xl mx-auto p-4 lg:p-6
  • Footer colle en bas grâce au flex column

Bonnes pratiques

Faire

  • Utiliser les classes DaisyUI (bg-primary, text-accent-content)
  • Utiliser les variables CSS (var(--color-primary))
  • Garder le CSS custom au strict minimum
  • Utiliser la structure .table pour les tableaux
  • Utiliser Roboto pour le corps, Nunito pour les titres
  • Tester avec data-theme="soliguide"

Ne pas faire

  • Hardcoder des couleurs hex (color: #f84b32)
  • Redéfinir les variables du thème dans un projet
  • Créer des styles CSS custom pour les tableaux
  • Utiliser des polices autres que Roboto/Nunito
  • Utiliser React, Vue, Svelte ou un framework JS
  • Utiliser text-gray-* (préférer text-base-content/60)
