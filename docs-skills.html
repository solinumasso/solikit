Comprendre les skills

Les commandes slash disponibles dans Claude Code pour accélérer le travail.

Les skills sont des commandes slash utilisables dans Claude Code. Chaque skill est un fichier .claude/skills/<nom>/SKILL.md qui guide l'assistant dans une tâche précise.

Tu tapes la commande, Claude pose les bonnes questions, puis construit tout automatiquement en respectant les conventions du projet.

Toutes les skills

Commande Quand l'utiliser Ce que ça fait
/pitch Tu arrives et tu as une idée Pose 3 questions, puis construit tout le projet automatiquement
/new-project Tu veux créer un nouveau projet Questions guidées → structure complète → pipeline → dashboard
/new-dashboard Tu as déjà des données JSON Crée ou refait le dashboard d'un projet existant
/process-data Tu veux traiter des données Crée un pipeline : téléchargement → transformation → export JSON
/csv-data-summarizer Tu as un CSV à explorer Analyse automatique : stats, types, distribution, aperçu
/frontend-design Tu veux un beau rendu Conseils de design frontend pour éviter l'esthétique "IA générique"
/writing-plans Tu veux planifier avant de coder Plan d'implémentation structuré, étape par étape
/skill-creator Tu veux créer une nouvelle skill Crée ou modifie un fichier SKILL.md

Comment ça marche

Les skills sont stockées dans .claude/skills/. Chaque dossier contient un SKILL.md qui décrit :

  • Les questions à poser à l'utilisateur
  • Les templates de code à générer
  • Les conventions à respecter (thème, structure, etc.)
  • Les vérifications à faire avant de livrer

Pour utiliser une skill, tape simplement la commande slash dans Claude Code :

# Dans Claude Code
/pitch Je veux analyser les bibliothèques de France et leur couverture Soliguide

Créer sa propre skill

Tu peux créer de nouvelles skills pour automatiser des tâches récurrentes. Utilise la commande /skill-creator ou crée manuellement un fichier :

# Structure d'une skill
.claude/skills/
└── ma-skill/
    └── SKILL.md       # Instructions pour Claude

Le fichier SKILL.md contient les instructions en langage naturel que Claude suivra quand la commande sera invoquée.
